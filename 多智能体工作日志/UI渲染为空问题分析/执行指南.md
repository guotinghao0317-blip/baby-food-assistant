# UI渲染为空问题修复 - 执行指南

## 快速开始

### 问题概述
**症状**: 数据加载成功，但页面显示为空

**根本原因**: `fetchRecipe`函数只更新了`days[].status='done'`，但没有填充`days[].details`字段，导致`getCurrentDayDetails()`返回undefined，UI无法渲染。

### 第3步: 添加重复数据检测（可选，用于诊断）

```typescript
const organizeRecipes = () => {
  if (!recipe || !recipe.details) {
    return {}
  }

  const organized: Record<number, Record<string, any>> = {}
  recipe.details.forEach((detail: any) => {
    if (!organized[detail.day_of_week]) {

    organized[detail.day_of_week][detail.meal_type] = detail

}
```

**修改后**:
```typescript
const organizeRecipes = () => {
  if (!recipe || !recipe.details) {
    return {}
  }

  const organized: Record<number, Record<string, any>> = {}
  const duplicates: string[] = []  // 新增：记录重复

  recipe.details.forEach((detail: any) => {
    if (!organized[detail.day_of_week]) {
      organized[detail.day_of_week] = {}
    }
    
    // 新增：检测重复
    if (organized[detail.day_of_week][detail.meal_type]) {
      duplicates.push(`第${detail.day_of_week}天-${detail.meal_type}`)
      console.warn(`⚠️ 发现重复数据: day=${detail.day_of_week}, meal=${detail.meal_type}, id=${detail.id}`)
    } else {
      organized[detail.day_of_week][detail.meal_type] = detail
    }
  })
  
  // 新增：输出重复信息
  if (duplicates.length > 0) {
    console.warn('⚠️ 检测到重复的餐次数据:', duplicates)
  }

  return organized
}
```
