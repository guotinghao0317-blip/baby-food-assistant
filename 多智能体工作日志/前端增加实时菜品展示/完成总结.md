## 核心实现思路

### 1. 数据流
```
后端 SSE 事件
  ↓
day_done 事件接收
  ↓
保存到 daysData 状态
  ↓
启动打字机动画
  ↓
更新渲染状态
  ↓
显示菜品列表
```

### 2. 状态管理
- 使用 React Hooks 管理状态
- daysData 存储完整数据
- selectedDay 控制当前显示
- renderedDishes 控制逐步动画

### 3. 动画实现
- 定时器更新状态
- React 重新渲染
- CSS 动画配合
- 实现流畅的打字机效果

### 4. 用户交互
- Tab 切换触发重渲染
- 重新启动打字机动画
- 提供良好的视觉反馈
