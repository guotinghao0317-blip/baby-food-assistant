## 检查清单

### 一、通用布局检查 (每个页面都要检查)

| 检查项 | 设备尺寸 | 预期结果 | 测试结果 |
|--------|----------|----------|----------|
| 是否出现水平滚动条 | 360px 手机 | 不应该出现 | ☐ 通过 ☐ 失败 |
| 内容是否贴边 | 所有手机 | 左右应有留白 (px-4) | ☐ 通过 ☐ 失败 |
| 容器是否居中 | 所有尺寸 | 内容区自动居中 | ☐ 通过 ☐ 失败 |
| 按钮可点击区域 | 所有手机 | 按钮高 ≥ 44px | ☐ 通过 ☐ 失败 |
| 文字大小 | 所有手机 | 正文字 ≥ 14px | ☐ 通过 ☐ 失败 |
| 横屏切换 | 手机 | 重新布局正确，不溢出 | ☐ 通过 ☐ 失败 |
| 点击元素间距 | 所有手机 | 可点击元素不挤在一起 | ☐ 通过 ☐ 失败 |
