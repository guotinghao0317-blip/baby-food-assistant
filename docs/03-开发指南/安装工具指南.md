## 🚀 方式一：使用安装脚本（推荐）

### 步骤

1. **打开终端**，进入项目目录：
```bash
cd /Users/jiayindeng/cursor尝试
```

2. **给脚本添加执行权限**：
```bash
chmod +x install-tools.sh
```

3. **运行安装脚本**：
```bash
./install-tools.sh
```

脚本会自动：
- 安装 Homebrew（如果没有）
- 安装 Python 3.11
- 安装 PostgreSQL 15
- 安装 Redis
- 提示 Docker 安装方法

**注意**：安装过程中可能需要输入管理员密码（sudo）。

## 🔧 方式二：手动安装

### 1. 安装 Homebrew（如果没有）

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

安装完成后，根据提示将 Homebrew 添加到 PATH。

### 2. 安装 Python 3.11

```bash
brew install python@3.11
```

验证：
```bash
python3.11 --version
```

### 3. 安装 PostgreSQL

```bash
brew install postgresql@15
brew services start postgresql@15
```

创建数据库：
```bash
createdb babyfood_db
```

验证：
```bash
psql --version
```

### 4. 安装 Redis

```bash
brew install redis
brew services start redis
```

验证：
```bash
redis-cli ping
# 应该返回: PONG
```

### 5. 安装 Docker（可选但推荐）

**方式 A：使用 Homebrew**
```bash
brew install --cask docker
```

**方式 B：从官网下载**
访问：https://www.docker.com/products/docker-desktop/
下载并安装 Docker Desktop

验证：
```bash
docker --version
```

## 🐛 常见问题

### Q1: Homebrew 安装失败
**解决**：
- 确保网络连接正常
- 可能需要代理（如果在中国大陆）
- 检查是否有管理员权限

### Q2: PostgreSQL 启动失败
**解决**：
```bash
# 检查状态
brew services list

# 手动启动
brew services start postgresql@15

# 如果端口被占用，检查并修改配置
ps aux | grep postgres
```

### Q3: Redis 启动失败
**解决**：
```bash
# 手动启动
redis-server

# 或使用 brew services
brew services start redis
```

### Q4: Python 版本问题
**解决**：
- 如果系统默认 Python 仍是旧版本，使用 `python3.11` 命令
- 或创建别名：
```bash
echo 'alias python3=python3.11' >> ~/.zshrc
source ~/.zshrc
```

## 📝 安装完成后的下一步

1. **配置后端环境变量**：
```bash
cd backend
cp .env.example .env
# 编辑 .env，填入 OPENAI_API_KEY
```

2. **启动服务**（使用 Docker）：
```bash
docker-compose up -d
```

或**手动启动**：
```bash
# 后端
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload

# 前端（新终端）
cd frontend
npm install
npm run dev
```
