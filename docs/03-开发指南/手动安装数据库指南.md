# 📦 手动安装数据库指南

由于安装 Homebrew 需要管理员权限，请按照以下步骤手动安装：

## 🚀 步骤 1：安装 Homebrew

在终端中运行以下命令（**需要输入管理员密码**）：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

安装过程中会：
1. 提示输入管理员密码（输入时不会显示，正常）
2. 自动下载并安装 Homebrew
3. 提示将 Homebrew 添加到 PATH

**如果安装成功，会看到类似提示：**
```
==> Next steps:
- Run these two commands in your terminal to add Homebrew to your PATH:
    echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
    eval "$(/opt/homebrew/bin/brew shellenv)"
```

**请按照提示执行这两条命令**（如果是 Intel Mac，路径可能是 `/usr/local/bin/brew`）

---

## 🐘 步骤 2：安装 PostgreSQL

安装完 Homebrew 后，运行：

```bash
# 安装 PostgreSQL 15
brew install postgresql@15

# 启动 PostgreSQL 服务
brew services start postgresql@15

# 创建项目数据库
createdb babyfood_db
```

验证安装：
```bash
psql --version
# 应该显示：psql (PostgreSQL) 15.x
```

---

## 🔴 步骤 3：安装 Redis

```bash
# 安装 Redis
brew install redis

# 启动 Redis 服务
brew services start redis
```

验证安装：
```bash
redis-cli ping
# 应该返回：PONG
```

---

## ✅ 步骤 4：验证所有工具

运行以下命令检查：

```bash
echo "Node.js: $(node --version)"
echo "Python: $(python3 --version)"
echo "PostgreSQL: $(psql --version)"
echo "Redis: $(redis-server --version)"
```

## 🐳 替代方案：使用 Docker（推荐）

如果不想手动安装，可以安装 **Docker Desktop**：

1. **下载 Docker Desktop**：
   - 访问：https://www.docker.com/products/docker-desktop/
   - 下载 macOS 版本
   - 安装并启动 Docker Desktop

2. **使用 Docker 启动所有服务**：
```bash
cd /Users/jiayindeng/cursor尝试
docker-compose up -d
```

Docker 会自动安装并启动 PostgreSQL 和 Redis，无需手动安装！

## ❓ 遇到问题？

### Q: Homebrew 安装失败
- 确保网络连接正常
- 可能需要代理（如果在中国大陆）
- 检查是否有管理员权限

### Q: PostgreSQL 启动失败
```bash
# 检查服务状态
brew services list

# 手动启动
brew services start postgresql@15

# 检查端口是否被占用
lsof -i :5432
```

### Q: Redis 启动失败
```bash
# 手动启动
redis-server

# 或使用 brew services
brew services start redis
```
